# Assets Directory

This directory is for placeholder images and static assets.

For this demo application, we use placeholder image URLs from via.placeholder.com and emoji icons.

## Recommended Assets

If you want to add custom images, place them here:

- `logo.png` - Application logo
- `avatar-default.png` - Default user avatar
- `food-placeholder.png` - Default food image
- `banner-hero.png` - Hero banner image
- `illustrations/` - Custom illustrations for banners and promos
